VesselHQ queue workspace. Add prepared posts, attach an image for each one (drag, click, or paste from clipboard), and save to the daily queue. Every new row starts with Facebook preselected; X / Twitter and LinkedIn can be added only from the optional setup panel. Same scheduled order = same cron run. Draft text is auto-saved locally.

Automatic Daily Publish Facebook only. Every new row starts with Facebook enabled, and the daily cron publishes only the next pending Facebook queue item.
Manual Platforms X / Twitter and LinkedIn are optional manual add-ons. They do nothing unless you enable them in the setup panel or trigger them from the queue.
Safety Guard Publish-now actions require confirmation, and saving a batch with optional X / LinkedIn enabled shows an extra warning.
